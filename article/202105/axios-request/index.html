axios的基本介绍

Axios 是一个基于 promise 的 HTTP 库，可以用在浏览器和 node.js 中。
axios支持请求的方式：

  • get：获取数据
  • post：提交数据（表单提交+文件上传）
  • put：更新数据（所有数据推送到后端）
  • patch：更新数据（只将更改的数据推送到后端）
  • delete：删除数据
